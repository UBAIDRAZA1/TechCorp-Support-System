## Common Issues & Solutions

### Login Issues
**Problem:** Cannot login to CRM
**Solution:** 
1. Check email/password
2. Reset password via "Forgot Password"
3. Clear browser cache
4. Contact support if issue persists

### Data Import
**Problem:** CSV import failing
**Solution:**
1. Check CSV format (must have headers)
2. Ensure file size < 10MB
3. Verify column mapping
4. Check for duplicate records

### Integration Setup
**Problem:** Email integration not working
**Solution:**
1. Verify email credentials
2. Check IMAP/SMTP settings
3. Enable less secure apps (Gmail)
4. Re-authorize OAuth connection

### API Access
**Problem:** API returning 401 errors
**Solution:**
1. Check API key validity
2. Verify endpoint URL
3. Check rate limits
4. Regenerate API key if needed
