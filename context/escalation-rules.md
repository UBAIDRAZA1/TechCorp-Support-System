# Escalation Rules

## When to Escalate

### Automatic Escalation (L2 - Human Support)
1. **Billing Issues**
   - Double charges
   - Refund requests
   - Payment failures
   - Invoice disputes

2. **Technical Issues**
   - Data loss
   - Security concerns
   - System outages
   - API failures

3. **Customer Sentiment**
   - Angry customers
   - Threats to cancel
   - Legal mentions
   - Executive escalation

4. **SLA Breaches**
   - Response time exceeded
   - Resolution time exceeded
   - Multiple follow-ups

### L3 Escalation (Technical Specialist)
1. **Complex Technical Issues**
   - Integration failures
   - Custom development needs
   - Performance issues
   - Data migration

2. **Feature Requests**
   - New feature requests
   - Customization needs
   - Enterprise requirements

3. **Product Feedback**
   - Detailed product feedback
   - Competitive analysis
   - Strategic input

### L4 Escalation (Engineering)
1. **Bugs**
   - Critical bugs
   - Security vulnerabilities
   - Data corruption
   - System-wide issues

2. **Infrastructure**
   - Server issues
   - Database problems
   - Network issues
   - Performance degradation

## Escalation Matrix

| Issue Type | Priority | Escalation Level | Response Time |
|------------|----------|------------------|---------------|
| Login Issues | Low | L1 | Immediate |
| Login Issues | High | L2 | 1 hour |
| Billing | Medium | L2 | 2 hours |
| Billing | High | L2 | 30 minutes |
| Technical Bug | Low | L3 | 4 hours |
| Technical Bug | High | L4 | 1 hour |
| Feature Request | Low | L3 | 24 hours |
| Security | High | L4 | Immediate |

## Sentiment-Based Escalation

| Sentiment Score | Action |
|-----------------|--------|
| Very Positive | No escalation, upsell opportunity |
| Positive | No escalation |
| Neutral | No escalation |
| Frustrated | Monitor, consider L2 |
| Angry | Escalate to L2 |
| Very Angry | Escalate to L2 + Priority |

## De-escalation Guidelines

1. **Acknowledge the issue**
2. **Apologize sincerely**
3. **Take ownership**
4. **Provide clear timeline**
5. **Follow up proactively**
6. **Offer compensation if appropriate**
